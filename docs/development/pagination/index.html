Pagination

One of the challenges when using graphql is pagination. Facebook's relay cursor pagination seems currently the best solution on implementing infinite scroll view which we normally see in many mobile apps.

Although it has some limitations when we face offset pagination scenario as above, relay cursor pagination is currently the most ideal way to support mobile's infinite scrollview.

Server#

Previously, implementing relay-cursor-pagination with orm like sequelize was a bit complicated. Here's how we've it done in medium post previously.

However, we've migrated to primsa2 and nexus so this becomes much easier. Nexus even supports connection plugin and we'll be using it in our server.

Client#

In client, we use relay. There is a great explanation on using relay pagination in react.
